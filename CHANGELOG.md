# Changelog

## v0.2.0

- Initial Python/PyQt6 GPLv3 package created from the SoundHelix source archive context.
- Added pure-Python MIDI writer.
- Added extended arrangement generator with track roles for drums, bass, chords, arpeggios, melody, pads, counter-melody and texture.
- Added GUI preset editor, track table, history, ratings and XML inspector.
- Added public-domain-inspired example presets plus an original arcade/popcorn-style hook preset.
- Added Windows installer and Nuitka compile script.
